1. PROBLÈME ACTUEL

Dans l’espace Agent Réservation :
- les informations s’affichent correctement

Mais dans l’espace Agent En Route :
- les informations ne sont pas toujours actualisées
- certaines données ne remontent pas en temps réel

---

2. TEMPS RÉEL (TRÈS IMPORTANT)

Toutes les données doivent être synchronisées en temps réel entre :
- client
- agent réservation
- agent en route
- entreprise

Quand une réservation est créée ou validée :
- elle doit apparaître immédiatement chez l’agent en route
- sans rechargement manuel
- sans délai

---

3. SUPPRIMER LES LIMITES (CRITIQUE)

Actuellement, il y a des limites (ex: 10 éléments affichés).

À corriger :
- ne pas limiter à 10
- ne pas limiter les listes
- afficher toutes les données disponibles

Cela concerne :
- réservations
- passagers
- départs
- colis

---

4. LOGIQUE COMPLÈTE À RESPECTER

Toujours appliquer la logique complète :

Quand un client réserve :
→ agent réservation reçoit
→ agent réservation valide
→ agent en route reçoit automatiquement

Quand agent en route ajoute un passager :
→ agent réservation voit
→ entreprise voit
→ historique mis à jour

Aucune rupture dans la logique.
